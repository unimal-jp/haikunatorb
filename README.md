# Haikunatorb

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'haikunatorb'
```

And then execute:
```ruby
bundle install
```


## Usage

```ruby
Haikunatorb.hex(5)
=> "steep-inu-7c3f6"

Haikunatorb.only_hex(4)
=> "6073"

Haikunatorb.num(3)
=> "patient-mannenhitsu-390"

Haikunatorb.only_num(6)
=> "903603"

Haikunatorb.alpha(5)
=> "wandering-mizu-smEPZ"

Haikunatorb.alnum(5)
=> "red-musashi-3gEpK"


```
